# COMP3011 Coursework 2: Search Engine Tool

A command-line search engine that crawls [quotes.toscrape.com](https://quotes.toscrape.com/), builds an inverted index with TF-IDF scoring, and provides interactive search functionality.

## Architecture Overview

```
cw2/
├── src/
│   ├── crawler.py    # Web crawler with politeness window
│   ├── indexer.py    # Inverted index builder, storage and loader
│   ├── search.py     # Search logic: print and find commands
│   └── main.py       # Interactive CLI shell entry point
├── tests/
│   ├── test_crawler.py
│   ├── test_indexer.py
│   ├── test_search.py
│   └── test_main.py
├── data/
│   └── index.json    # Compiled inverted index (generated by build)
├── requirements.txt
└── README.md
```

### Design Decisions

- **Inverted Index**: A nested dictionary mapping each word to a posting list — for every URL it records `{frequency, positions, tf, tfidf}`. This enables O(1) word lookup and O(n) set intersection for multi-word queries.
- **TF-IDF Ranking**: `tf(w, d) = freq(w, d) / total_tokens(d)` and `idf(w) = log(indexed_pages / df(w))`. Results from `find` are ranked by summed TF-IDF so the most relevant pages appear first.
- **Storage Format**: JSON — human-readable, easily inspectable, and requires no extra dependencies. The `__meta__` key stores build statistics (`total_pages`, `indexed_pages`, `total_words`, `built_at`).
- **Politeness Window**: A hard 6-second sleep between every HTTP request to respect the target server.
- **Two-Phase Crawl**: Phase 1 follows pagination links across quote listing pages; Phase 2 visits author detail pages discovered in Phase 1. Both phases share the same politeness window.
- **Stop Words**: Common English stop words are excluded from the index to reduce noise and improve query relevance.

## Installation

```bash
# Clone the repository
git clone https://github.com/wwbotww/comp3011-web-cw2.git
cd comp3011-web-cw2

# (Recommended) Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Usage

Start the interactive shell:

```bash
python src/main.py
```

### Commands

| Command | Description | Example |
|---------|-------------|---------|
| `build` | Crawl the website and build the inverted index | `> build` |
| `load` | Load a previously built index from disk | `> load` |
| `print <word>` | Display the inverted index entry for a word | `> print nonsense` |
| `find <query>` | Find all pages containing the query term(s) | `> find indifference` |
| `help` | Show available commands | `> help` |
| `quit` / `exit` | Exit the program | `> quit` |
